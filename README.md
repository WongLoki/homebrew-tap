# Homebrew Tap

Custom Homebrew formulae by WongLoki.

## Install

```bash
brew tap WongLoki/tap
brew install kakuai
```

## Formulae

| Formula  | Description                                                           |
| -------- | --------------------------------------------------------------------- |
| `kakuai` | AI coding tools configuration TUI for Claude Code, OpenCode, OpenClaw |

## Usage

```bash
kaku ai
```
